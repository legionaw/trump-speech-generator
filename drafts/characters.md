# Character data #

This file contains the character data for the generator. It is used to define the characters that will be available in the generator, including their names, descriptions, and other relevant information.

## Adding New Characters ##

To ensure variety, pre-populate each phase (Q&A, social media, round-table) with more characters than the maximum number that will appear in any single run. This allows Gemini to select a different set of characters for each phase, increasing realism and diversity.
